# Retinoblastoma Detection System

![Project Banner](docs/banner.png)  <!-- optional banner image -->

## Overview
This project is a **Retinoblastoma Detection System** that classifies retinal images as either **Normal** or **Retinoblastoma Detected**. It leverages deep learning models (EfficientNet-B0) trained on retinal datasets and provides an **easy-to-use web interface using Streamlit** for image upload and prediction.

Retinoblastoma is a rare eye cancer that typically affects young children. Early detection is crucial to preserve vision and prevent severe complications.

---

## Features

- Upload a retinal image through a web-based frontend.
- Predict whether the image shows signs of Retinoblastoma.
- Display results clearly with visual feedback.
- Supports an **ensemble of pre-trained models** for more reliable predictions.
- CPU-friendly implementation for wider accessibility.
